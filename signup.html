Public Access Unix System (pubnix)

OSLI will be setting up a pubnix, a Unix system onto which anyone (who has an account) can connect, host web pages, run gopher and gemini sites, chat with other members of the system, and play terminal-based games.

The system is not running yet (aiming for before September 2023), but a signup link is available so that you can be a member as soon as the network is running.
